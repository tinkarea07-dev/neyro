▶ Где взять токен?

1. Открой эту страницу GitHub

2. Note: Nik Admin

3. Expiration: 90 days (или больше)

4. Scope: ✅ repo (полный доступ к репо)

5. Generate token → скопируй и вставь сюда

Токен хранится только в твоём браузере (localStorage), никуда не отправляется.
