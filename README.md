# Telegram Бот для Нейрокомментинга

Полнофункциональный бот для автоматического комментирования постов в Telegram каналах с использованием GPT-5 через ProxyAPI.

## 🚀 Функционал

### ✅ Реализовано:
1. **Комментирование постов** - автоматическая генерация комментариев через GPT-5
2. **Автоматическое вступление в каналы** - из списка с задержками
3. **Автоответчик** - ответы на личные сообщения от лица девушки-аниме фаната
4. **Поиск по ключевым словам** - поиск каналов и постов
5. **Чекер комментариев** - проверка возможности комментирования

## ⚙️ Установка и настройка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Отредактируйте файл `1.envv`:

```env
OpenAI_token=sk-YyLRQLcuUisIOZbDzXj0XEaoRtxOVIIl
Api_id=your_api_id
Api_hash=your_api_hash
Owner_id=@your_username
```

**Где получить api_id и api_hash:**
- Перейдите на https://my.telegram.org/apps
- Войдите со своим номером телефона
- Создайте приложение
- Скопируйте `api_id` и `api_hash`

**Или используйте скрипт:**
```bash
python create_session_smart.py
```

### 3. Создание сессии Telegram

```bash
python create_session_smart.py
```

Введите номер телефона и код подтверждения из Telegram.

### 4. Настройка списка каналов

Отредактируйте `data/channels_to_join.txt` - добавьте каналы для автоматического вступления:

```
@channel1
@channel2
channel3
```

### 5. Настройка ключевых слов

Отредактируйте `data/keywords.txt`:

```
аниме
новости
технологии
```

## 🔧 Конфигурация

Основные настройки находятся в `config.py`:

- `COMMENT_DELAY_MIN/MAX` - задержка между комментариями (сек)
- `JOIN_CHANNEL_DELAY_MIN/MAX` - задержка между вступлениями (сек)
- `AUTORESPONDER_ENABLED` - включить/выключить автоответчик
- `AUTORESPONDER_MIN_MESSAGES_BEFORE_PROMOTE` - после скольких сообщений рекламировать канал
- `SEARCH_ENABLED` - включить/выключить поиск
- `GIRL_PERSONA_SYSTEM` - промпт для персоны девушки-аниме фаната

## 📝 Как работает

### Комментирование:
1. Бот мониторит список активных каналов
2. При появлении нового поста генерирует комментарий через GPT-5
3. Отправляет комментарий с задержкой
4. Отправляет отчет владельцу

### Автоответчик:
1. Прослушивает входящие личные сообщения
2. Проверяет, что пользователь незнакомый
3. Генерирует ответ через GPT-5 с персоной девушки
4. После 3+ сообщений естественно упоминает аниме-канал из описания профиля

### Вступление в каналы:
1. Загружает список каналов из `data/channels_to_join.txt`
2. Вступает в каждый канал с задержками
3. Отправляет отчет о результатах

### Поиск:
1. Ищет каналы по ключевым словам
2. Ищет посты в каналах по ключевым словам
3. Автоматически добавляет найденные каналы в список

### Чекер:
1. Проверяет каждый канал на возможность комментирования
2. Фильтрует только активные каналы
3. Сохраняет статус в `channel_status.json`

## 🐛 Устранение неполадок

### Ошибка "ApiIdInvalidError":
- Проверьте правильность api_id и api_hash
- Используйте `create_session_smart.py` для создания сессии

### Бот не комментирует:
- Проверьте, что каналы публичные
- Убедитесь, что в каналах включены комментарии
- Проверьте логи на ошибки

### Автоответчик не работает:
- Проверьте, что `AUTORESPONDER_ENABLED = True` в config.py
- Убедитесь, что сообщения от незнакомых пользователей
- Проверьте логи

## 📞 Поддержка

При возникновении проблем:
1. Проверьте логи в `data/bot.log`
2. Проверьте настройки в `config.py`
3. Убедитесь, что все зависимости установлены
4. Проверьте переменные окружения в `1.envv`
